# Discourse BEditor plugin
Replacing Discourse Editor (d-editor) to Better Editor with WYSIWYG feature.

## Development
You must have `node` and `npm` for development:

```
# Installing dependencies
$ npm install

# Do this everytime you make changes
$ npm run build 
```
